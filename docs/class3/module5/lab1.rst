UCS, BIG-IP Archive
===================

Create UCS Archive Files
------------------------

Open **System > Archives** page.

Create new archive named **backup\_labs\_1\_to\_4**

.. admonition:: TMSH

   You can also create backups via TMSH: 
   **sav sys ucs <filename>**
